Amazon Movie Review Analysis

Amazon Movie Review Analysis provides a web interface integrated with AWS SDK to dynamically create Hadoop clusters on AWS and read files from S3. It uses the Amazon Movie Review Dataset to identify top K helpful reviewers and provide movie recommendations based on the genre and type of rating using MapReduce programming paradiagm.
